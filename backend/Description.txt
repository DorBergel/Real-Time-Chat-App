# Current APIs Implemented in Backend Project

1. **User Authentication**
    - `POST /api/auth/register` - Register a new user
    - `POST /api/auth/login` - Log in an existing user

2. **User**
    - `GET /api/user/contacts/:userId` - Get user contacts
    - `GET /api/user/chats/:userId` - Get user chats
    - `GET /api/user/username/:userId` - Get user username

3. **Chat**
    - `GET /api/chat/:chatId` - Get chat details

4. **Messages**
    - `GET /api/message/:chatId` - Get chat messages
    - `POST /api/message/:chatId` - Create a new chat message
